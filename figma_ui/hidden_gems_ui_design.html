05 — Motion

Animation System

All animations use flutter_animate. Orchestrated reveals, not scattered effects. Motion is ambient — the app breathes.

🌊
Page Entrance
Cards slide up with staggered fade-in using AnimationConfiguration.staggeredList. Each item adds 100ms delay.
fadeIn(600ms) + slideY(50px → 0)
🌿
Oracle Orb Pulse
The FAB orb gently pulses with a radial glow. Scale 1.0 → 1.06 → 1.0 on loop, synchronized to a 2.5s cycle.
scale loop(2500ms) + glow shimmer
🔄
Hero Image Crossfade
Background images on HomeScreen rotate every 4 seconds with AnimatedSwitcher crossfade at 1500ms.
AnimatedSwitcher(1500ms crossfade)
✨
Premium Shimmer
Heritage and premium cards shimmer on first appear after 2s delay, drawing attention without being disruptive.
shimmer(delay: 2s, 1500ms)
📍
Detail Stagger
Place Details sections load in sequence: header → AI reason → stats → details. Each +200ms delay from previous.
fadeIn + slideX, 200ms stagger
🎯
Haptic Feedback
All nav taps and confirmations use HapticFeedback.lightImpact(). Errors use mediumImpact. Never heavy unless destructive.
lightImpact on tap · mediumImpact on error

07 — Rules

Design Principles

🌑
Natural Dark First

Background is #080B0F — not pure black (#000). This gives depth without harshness. Never use black fills or harsh contrast. The darkness feels organic, like being in a jungle at night.

🌿
Green is Earth, Not Electric

#22C55E is a leaf green — rich, alive, not fluorescent. Glow is always subtle: rgba(34,197,94,0.15) max. Primary actions only. Secondary elements use the tint, never the full green.

🪟
Glass Sparingly

Glassmorphism only for Oracle AI cards, hero overlays, and floating elements. Standard cards use solid #141C24. Too much glass = no depth. Glass is reserved for "magical" moments.

📐
Border Radius System

Small elements: 12px. Cards: 18–24px. Buttons: pill (100px). Avatars: full circle. Consistency is identity — never mix arbitrary radius values within the same screen.

🔤
Typography Hierarchy

Outfit 900 for hero titles (UPPERCASE + letter-spacing). Outfit 700–800 for card titles. Inter 400–500 for body. NEVER use Inter for headings or Outfit for body copy. Each font has one role.

✋
One Primary Action

Each screen has exactly one primary green CTA. Secondary actions use glass or ghost style. Gold is exclusively for premium/heritage CTAs. Never two green buttons on the same screen.
